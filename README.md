# Set Up Flask Application
- cd api
- .\env\Scripts\activate 
- pip install flask
- pip install python-dotenv
- pip install scikit-learn==1.3.0
- pip install tensorflow==2.13.0

#### For Set Debuger On
- FLASK_APP=app.py
- FLASK_ENV=development

## Run Flask Application 
- flask run

### New Command Prompt (Run React)
- cd ..
- npm run dev
