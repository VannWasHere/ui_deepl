# React + Flask 

How to run project:
- npm install
- cd api
- .\env\Scripts\activate 
- pip install flask
- pip install python-dotenv
- pip install scikit-learn==1.3.0
- pip install tensorflow==2.13.0
- FLASK_APP=app.py
- FLASK_ENV=development

## Run Application
- flask run
- cd ..
- npm run dev
